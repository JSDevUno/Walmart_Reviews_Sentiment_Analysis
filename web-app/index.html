About Walmart Sentiment Analyzer

Version: 1.0.0

Purpose: This tool analyzes customer reviews from Walmart products and classifies them into positive, negative, or neutral sentiments using machine learning.

How to Use:

  1. Copy a Walmart product URL (e.g., walmart.com/ip/product-name/123456)
  2. Paste it into the input field
  3. Click "Analyze Reviews"
  4. Wait for the analysis to complete (may take 1-2 minutes)
  5. View the sentiment distribution and sample reviews

Features:

  • Automated web scraping of Walmart reviews
  • Machine learning-based sentiment classification
  • Visual sentiment distribution charts
  • Sample reviews for each sentiment category
  • Rating statistics and distribution

Technical Details:

Backend: Python, Flask, Selenium, scikit-learn

Frontend: HTML5, CSS3, JavaScript

Model: LinearSVC with TF-IDF vectorization

Limitations:

  • Limited to 5 pages of reviews per analysis for performance
  • Requires active internet connection
  • May encounter CAPTCHA challenges

Created for sentiment analysis research and educational purposes.
