About This Tool

Walmart Sentiment Analyzer

This tool uses machine learning to analyze customer reviews from Walmart products and determine their sentiment.

Features:

  • Automated web scraping of Walmart product reviews
  • TF-IDF + SVM sentiment classification (Positive, Negative, Neutral)
  • Confidence scores for each prediction
  • Statistical analysis and visualization
  • Sample reviews from each sentiment category

How to Use:

  • Copy a Walmart product URL from walmart.com
  • Paste it into the input field
  • Click "Analyze Reviews" and wait for results
  • View comprehensive sentiment analysis and sample reviews

Technology:

Built with Python (Flask, Selenium, scikit-learn), HTML, CSS, and JavaScript. Uses TF-IDF vectorization and Linear SVM for sentiment classification.
